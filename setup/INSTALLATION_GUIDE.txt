📋 QUICK INSTALLATION STEPS
═══════════════════════════════════════════════════════════════════════════

STEP 1: UPLOAD FILES
────────────────────────────────────────────────────────────────────────────
1. Upload the 'setup' folder to your web server
2. Place it in your web root directory (e.g., /var/www/html/)

   Example structure:
   /var/www/html/
   └── setup/
       ├── index.php
       ├── install.php
       └── ...


STEP 2: SET PERMISSIONS
────────────────────────────────────────────────────────────────────────────
Run these commands on your server:

   chmod -R 755 setup/
   chmod -R 777 ../


STEP 3: PREPARE DATABASE
────────────────────────────────────────────────────────────────────────────
Create a database for N8N:

   MySQL/MariaDB:
   mysql -u root -p
   CREATE DATABASE n8n;
   CREATE USER 'n8n_user'@'localhost' IDENTIFIED BY 'your_password';
   GRANT ALL PRIVILEGES ON n8n.* TO 'n8n_user'@'localhost';
   FLUSH PRIVILEGES;
   EXIT;


STEP 4: ACCESS INSTALLER
────────────────────────────────────────────────────────────────────────────
Open your web browser and go to:

   http://your-domain.com/setup/

Or if on localhost:

   http://localhost/setup/

3. ✅ Database Setup - Enter credentials

🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

Issue: "PHP version too old"
Solution: Upgrade PHP to 7.4+
   sudo apt install php7.4  # Ubuntu/Debian
   sudo yum install php74   # CentOS/RHEL


Issue: "Node.js not found"
Solution: Install Node.js 18+
   curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
   sudo apt install -y nodejs


Issue: "Permission denied"
Solution: Fix file permissions
   sudo chmod -R 755 /path/to/installation
   sudo chown -R www-data:www-data /path/to/installation


Issue: "Database connection failed"
Solution: Check credentials and database existence
   - Verify username and password
   - Ensure database exists
   - Check host and port settings
